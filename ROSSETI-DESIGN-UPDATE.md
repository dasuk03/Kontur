# Обновление дизайна «Контур»

Интерфейс переведён на визуальную систему в стиле предоставленного референса:

- крупная типографика и короткие заголовки;
- мягкий светло-серый фон;
- округлые карточки и pill-кнопки;
- акцентные плитки для статистики;
- мобильная нижняя навигация;
- палитра Россетей: глубокий синий, голубой, белый и красно-оранжевый акцент;
- сохранены текущие сценарии: карта, реестр ТП, дерево ТП → фидер → опора → ПУ, чек-лист, локальное сохранение и экспорт Excel.

Изменённые файлы:

- `src/main.jsx` — добавлена подпись Россетей в шапке, мобильная навигация и светлая тема Telegram WebApp.
- `src/styles.css` — добавлена тема `Rosseti visual system` поверх существующих компонентов.

Проверка: `npm run build` проходит успешно.
